## Calendar Integration

The Events page renders Tiffany's public Google Calendar from its public iCal feed and styles the
events inside the site instead of using Google's iframe. The default feed is:

```bash
https://calendar.google.com/calendar/ical/bourbonattiffanys%40gmail.com/public/basic.ics
```

You can override it with a server-only environment variable:

```bash
GOOGLE_EVENTS_ICS_URL=
```

Calendar data is fetched server-side and revalidated every 6 hours. The site filters Google Calendar's
generic `Busy` placeholders, so only entries with public titles/details render. The Picks page does not
publish pick-trip travel dates; it remains focused on release and archive information.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
